Root element
------------
Specifies the name for the exported XML document's root element. By default, this value is "data".

### Example: ###

**Sheet name: `Colors`**

Name | R | G | B
---- | - | - | -
Red | 1 | 0 | 0
Green | 0 | 1 | 0
Blue | 0 | 0 | 1

*Default Root Element:*
```xml
<data>
  <Colors>
    <Red R="1" G="0" B="0"/>
    <Green R="0" G="1" B="0"/>
    <Blue R="0" G="0" B="1"/>
  </Colors>
</data>
```

*Root Element "Art":*
```xml
<Art>
  <Colors>
    <Red R="1" G="0" B="0"/>
    <Green R="0" G="1" B="0"/>
    <Blue R="0" G="0" B="1"/>
  </Colors>
</Art>
```
