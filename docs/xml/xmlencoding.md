XML encoding
------------
Specifies the encoding shown in the included XML declaration.

If set to `None`, encoding will not be included in the XML declaration.

Supported encodings |
------------------- |
UTF-8
UTF-16
ISO-10646-UCS-2
ISO-10646-UCS-4
ISO-8859-1
ISO-8859-2
ISO-8859-3
ISO-8859-4
ISO-8859-5
ISO-8859-6
ISO-8859-7
ISO-8859-8
ISO-8859-9
ISO-2022-JP
Shift_JIS
EUC-JP
